# Instruções para Claude Code

Você está desenvolvendo o ZOI Agent Framework.

Objetivo:
Construir uma arquitetura operacional escalável
para agentes comerciais orientados por IA.

Princípios:
- modularidade
- observabilidade
- desacoplamento
- persistência
- multi-tenant
- tool-based architecture

Sempre:
- leia os documentos em /docs
- siga architecture-rules.md
- siga coding-standards.md
- utilize as skills necessárias

Nunca:
- criar lógica acoplada
- misturar regras comerciais no prompt
- acessar integrações diretamente
- criar código sem tipagem
