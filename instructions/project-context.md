# Contexto do Projeto

A ZOI desenvolve agentes comerciais inteligentes
para concessionárias, energia solar e seguros.

O objetivo do framework é criar uma arquitetura
padronizada para agentes de IA comerciais.

Os agentes:
- qualificam leads
- consultam estoque
- agendam visitas
- realizam handoff humano
- atualizam CRM
- utilizam memória persistente

Lucas SDR Agent da AMC Veículos.
